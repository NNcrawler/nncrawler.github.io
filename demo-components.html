How to Use in Hugo

To use these components in your Hugo site:

  1. Add {{ partial "components.html" . }} to your template
  2. Use the custom elements in your HTML: <hello-world name="Reader"></hello-world>
  3. For portfolio: Use <portfolio-grid> with <project-item> children for SEO
  4. Components work in both templates and markdown (if raw HTML is enabled)
